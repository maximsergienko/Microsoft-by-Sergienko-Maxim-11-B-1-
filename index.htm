Основні дати в історії компанії

1978

Друзі по навчанню Білл Гейтс та Пол Ален засновали компанію Microsoft. Першим їх продуктом був інтерпритатор мови Basic.

1980

Microsoft підписала договір з ІВМ на розробку операційно системи MS-DOS.

1983

Реліз першої версії MS Word.

1985

Реліз першої версії Excel. Версія була призначена тільки для комп'ютерів Macintosh.

1995

Реліз Windows 95 з оновленим інтерфейсом та кнопкою "Пуск".

1995

Білл Гейтс став найбагатшою людиною в світі та зберігав цей статус до 2017 року.
